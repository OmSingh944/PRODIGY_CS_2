
# PRODIGY_CS_02

## Image Encryption Tool

This project is developed as part of the Prodigy InfoTech Cyber Security Internship.

### Description
This Python program encrypts and decrypts images using pixel manipulation techniques. The user can provide a secret key to encrypt an image and use the same key to decrypt it back to its original form.

### Features
- Encrypt image using a secret key
- Decrypt encrypted image
- Supports common image formats
- Simple command-line interface

### Technologies Used
- Python
- Pillow (PIL)
- NumPy
